# Book Store

An online bookstore web application built with **Python** and **Django**.  
The project demonstrates core web development skills: authentication, shopping cart, order management, and payment integration via Stripe.

---

## Features
- User registration and authentication (login/logout)
- Shopping cart functionality
- Order creation and management
- Stripe integration for online payments
- Optimized queries using Django ORM
- Templates for frontend pages

---

## Tech Stack
- **Backend:** Python, Django, Django ORM
- **Database:** SQLite (default, can be replaced with PostgreSQL/MySQL)
- **Frontend:** HTML, CSS
- **Tools:** Git, GitHub

---

## Project Structure
- `mysite/` — main Django project configuration  
- `orders/` — app handling cart and order logic  
- `templates/` — HTML templates for pages  
- `manage.py` — Django management script  
- `Pipfile` / `Pipfile.lock` — dependencies management
